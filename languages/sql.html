SQL

SQL (Structured Query Language) is a standard language for accessing and manipulating data held in relational databases. There are many different versions of SQL, though they all have basic similarities and support the same major commands to be compliant with the ANSI standard.

What SQL is good at

  • Quick and easy access to databases and manipulation of their data using simple commands
  • Management and administration of databases, including permissions, backup, restore, etc.
  • Fairly easy to learn with commands and keywords that read very much like English and common sense
  • Very useful for retrieving data for business intelligence, which makes a huge difference between meaningful and meaningless data for business purposes
  • Once mastered, very powerful and could also even be used as a full-blown programming language with application logic built into its procedures and functions

SQL's disadvantages and limitations

  • Can be unappealing to beginners, especially when there is no real data to work with and/or no real-life scenarios to show SQL's usefulness
  • Difficult to master and complex SQL code can be harder to understand than other languages
  • Quite fragmented due to different versions and implementations, which include proprietary features used by some vendors to maintain exclusivity
