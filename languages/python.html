Python

Python is an open-source, high-level general-purpose programming language whose popularity has been constantly increasing over the past few years. It often tops many lists of recommended programming languages to learn, generating a lot of attention on the topic, and for good reasons.

Why Python is so popular

  • It's very easy to learn due to its design philosophy that focuses on code readability using significant whitespace, as well as syntax that reads a lot like plain English. Therefore, it's often recommended as the first programming language for beginners to learn.
  • It has cross-platform support, so coders get the freedom of choosing their own system to have Python up and running on.
  • It has huge community support with tons of libraries, making it capable for a wide variety of different projects.
  • It's very flexible and supports multiple programming paradigms (procedural, functional, object-oriented, etc.), again giving choices for different types of developers and/or projects.
  • Python is used by many big name tech companies (Google, Facebook, Netflix, Dropbox, etc.), adding to it the cool factor, as well as increased marketability for Python developers.

Python's limitations

  • Partly due to its nature of being an interpreted language, Python's performance is generally slower than other compiled languages like Java or C++.
  • Surprising lack of support for mobile development.
  • Not a great choice for game development, especially graphics-intensive games.

What Python is used for

As mentioned above, Python is one of the best general-purpose languages that can be used for lots of different projects, from scripting to web development, desktop application development, etc. One of the most popular and perhaps most important uses of Python is for AI/machine learning, which has been a major focus of several tech giants over the past few years to work hand-in-hand with IoT (Internet of Things), which is also well-supported by Python.
